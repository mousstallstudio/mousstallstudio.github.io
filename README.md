## 🚀 Spécifications Techniques

Le projet est conçu pour être ultra-léger et performant (optimisé pour les configurations type **Acer Aspire ES1**).

- **Frontend :** HTML5 / CSS3 (Variables CSS pour le thème Néon) / Vanilla JS.
- **Design :** Mobile-First, responsive, et typographie `Fira Code`.
- **Infrastructure :** Hébergé sur GitHub Pages.
- **Sécurité :** Audit de code statique pour garantir l'absence de scripts malveillants.
